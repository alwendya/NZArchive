19NZAʀᴄʜɪᴠᴇ is a fast and encrypted archive format using ZStandard for fast compression
20and blazing fast decompression no matter the compression level.
21Encryption is using 20 round Chacha 256 bits or AES 256 buts.
22Written in C++17 and oriented to 64 bits application.
23
24
25▄ External library used ▄
26-------------------------
27 - ZSTD : ZStandard compression
28 - Bcrypt : Windows NextGen compression library (present starting Windows Vista, can be replaced by TinyAES if needed)
29
30▄ Using this internal library ▄
31-------------------------------
32 - Wirehair : Library for generating code recovery for incomplete file
33 - XXHASH : Hashing library
34
35▄ Revisionning ▄
36----------------
37 Version 1.5.0 : Optimization of Wirehair function inside NZArchive.
38 Version 1.4.0 : Integration of Wirehair to create receovery file.
39 For each 32MiB, a percentage of extra block is created and allow
40 to recover damaged part of the 32Mib block.
41 Version 1.3.0 : Modification of TinyAES256 Encryot/Decrypt function in CBC Mode
42 to ensure compatibility with Archive using BCrypt (And the invert too).
43 by creating a padding method to ensure block s a multiple of 16 bytes
44 (Standard definition of AES256 block size)
45 Version 1.2.0 : Added AES256 support with BCrypt
